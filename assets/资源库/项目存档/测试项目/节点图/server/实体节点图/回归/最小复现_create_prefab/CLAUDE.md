## 目录用途
存放 `测试项目` 的最小复现节点图（Graph Code）：聚焦 **创建元件（Create_Prefab）** 的最小链路，便于做端口索引/写回/导入口径的快速回归。

## 当前状态
- `TS_最小复现_创建元件.py`：实体创建触发 → 读取图变量元件ID → 创建元件（位置/旋转常量，拥有者实体为事件源实体），并提供 `__main__` 自检入口。

## 注意事项
- 该目录属于测试资产，只应存在于 `assets/资源库/项目存档/测试项目/` 范围内。
- 修改后建议跑：`python -X utf8 -m app.cli.graph_tools validate-file <节点图路径>`。
