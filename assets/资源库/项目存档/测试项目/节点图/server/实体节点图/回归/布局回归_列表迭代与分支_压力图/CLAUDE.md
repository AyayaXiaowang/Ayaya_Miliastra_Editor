# 布局回归_列表迭代与分支_压力图

## 目录用途

存放 `测试项目` 下“布局回归：列表迭代循环 + 分支压力”节点图（Graph Code，server/实体节点图）。

## 当前状态

- 包含 1 张压力回归图：`布局回归_列表迭代与分支_压力图.py`
  - 密集构造多个 `for <item> in <列表变量>`（列表迭代循环）；
  - 循环体内混合双分支（if/else）与多分支（match-case）；
  - 循环完成后仍有后续流程，确保同时存在 `循环体` 与 `循环完成` 两条出口链路，便于验证块编号不变量与排版确定性。

## 注意事项

- 本目录样本仅用于回归与诊断，不作为业务项目维护。
- 修改后必须校验：`python -X utf8 -m app.cli.graph_tools validate-file <文件路径>`。
- 文件名不要以下划线开头（否则 `validate-graphs --all` 默认收集会跳过）。
