## 目录用途
存放“布尔值常量/端口映射”的写回回归节点图（Graph Code）：覆盖常见带布尔输入的执行节点，确保 `.gil` 写回后布尔值不翻转、不丢失、不发生端口错位。

## 当前状态
- `TS_写回回归_布尔值端口映射.py`：实体创建触发 → 对多类节点分别写入 `True/False` 两组常量（含 `设置自定义变量/设置节点图变量` 等），用于导出/写回后对照 `.gil payload` 的 InParam(Bol) pins。
- `TS_写回回归_布尔值端口映射.py`：补齐【设置聊天频道开关】的必填输入端口 `语音开关`，确保 `validate-file` 与 UI 严格加载可通过。

## 注意事项
- 本目录只用于回归与诊断；不作为业务项目维护。
- 修改后建议跑：`python -X utf8 -m app.cli.graph_tools validate-file <节点图路径>`（单文件）。
- 该目录图的设计目标是“结构与常量覆盖”，不追求运行期语义（节点可能是空实现/仅日志）。
