## 目录用途
存放项目存档 `测试项目` 的管理配置资源（信号、结构体定义、关卡变量等），供资源索引/校验/导出/写回链路读取。

## 当前状态
- 当前主要包含 `信号/`：为测试集的“复合内发送信号/宿主图监听信号”提供白名单与参数类型定义。
- 包含 `UI源码/`：用于 UI 静态约束校验与 Web UI bundle 导入写回链路回归。

## 注意事项
- 每个资源子域（如 信号/结构体定义/关卡变量）使用独立子目录并维护自己的 `CLAUDE.md`。
- 修改后建议跑：`python -X utf8 -m app.cli.graph_tools validate-project --package-id 测试项目`。
