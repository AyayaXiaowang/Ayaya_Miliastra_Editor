## 目录用途
存放项目存档 `示例项目_第七关` 的 server 实体节点图（Graph Code `.py`）：挂在实体上运行，联动 UI/实体/信号。

## 当前状态
- 本目录仅保留 7 个示例图（需求指定的最小集合），用于覆盖“选关 → 游戏中 → 门流程/亲戚数据服务 → 结算推进”的关键链路。

## 注意事项
- 节点图修改后必须校验：`python -X utf8 -m app.cli.graph_tools validate-file <节点图路径>`。
- 跨图共享状态必须落在关卡实体自定义变量中，避免使用各图独立的 `GRAPH_VARIABLES` 造成不同步。
- 强类型列表端口（如 `浮点数列表/整数列表/字符串列表`）在校验阶段会逐元素检查类型；列表字面量会被归一化为 `拼装列表`，因此应在拼装前显式保证元素类型匹配（必要时用【数据类型转换】节点）。
