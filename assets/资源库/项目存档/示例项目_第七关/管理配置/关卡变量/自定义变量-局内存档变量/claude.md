## 目录用途
局内存档变量（Ingame Save Variables）代码资源根：仅供玩家实体使用，用于描述“局内存档/Profile”相关的可变状态。每个文件以 `VARIABLE_FILE_ID` 作为整组变量的稳定引用键。

## 当前状态
- 本目录仅放置“局内存档变量”；普通关卡变量位于同级 `自定义变量/`。
- 变量以 Python 代码级常量定义为准，供 Schema 视图与校验读取。

## 注意事项
- 变量文件必须导出 `VARIABLE_FILE_ID`、`VARIABLE_FILE_NAME`、`LEVEL_VARIABLES`。
- `variable_name` 必须遵循 `玩家槽位_chip_序号`（如 `1_chip_1`），同一槽位内序号从 1 开始连续递增。
- 仅局内存档变量允许引用存档结构体；普通自定义变量禁止引用存档结构体。
- 存档结构体字段禁止嵌套结构体/结构体列表；复杂子对象应拆分为独立槽位并通过 ID 字段关联。
