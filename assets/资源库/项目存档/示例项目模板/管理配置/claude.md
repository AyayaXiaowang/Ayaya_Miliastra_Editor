## 目录用途
- 存放“示例项目模板”的管理配置资源（JSON/Python 代码资源）：关卡变量、结构体定义、信号、计时器、预设点、货币背包等。
- 目录按资源类型分子目录组织（目录即归属），供资源索引/管理面板/校验链路统一读取。

## 当前状态
- 管理配置资源已按常用类型目录拆分（`关卡变量/`、`结构体定义/`、`信号/` 等），用于作为新建项目存档的复制模板。
- UI 工作流仅保留 **源码目录骨架**：
  - `UI源码/`：HTML UI 源码输入（一个 HTML 一个功能页，HTML 内通过 `{{lv.xxx}}` 与 `data-ui-key` 声明绑定与控件键）

## 注意事项
- 本目录以“管理配置资源”为主：绝大多数为 `*.json`，少量为代码级资源 `*.py`（如 `信号/`、`结构体定义/`、`关卡变量/`）。
- UI 派生物（如 ui_key→guid 注册表、UI 交互映射、HTML bundle 等）**统一写入运行时缓存** `app/runtime/cache/ui_artifacts/<package_id>/`，不应出现在资源库内。
- 保持目录结构稳定；不要在本目录写入运行时缓存或临时产物（缓存统一落在 `app/runtime/cache/`）。
