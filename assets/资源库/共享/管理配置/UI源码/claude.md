## 目录用途
共享 UI 源码目录：存放可跨多个项目复用的 **HTML UI 源码**（静态展示稿），供“千星沙箱网页处理工具 Workbench”直接浏览/预览，并可复制到项目存档后再导入为 UI 布局资源。

## 当前状态
- 目录由 Workbench 统一读取：与项目侧 `assets/资源库/项目存档/<package_id>/管理配置/UI源码/` 一起展示在“UI源码浏览器”中。
- 共享目录内的 HTML 视为 **只读模板/示例**：Workbench 默认不允许直接覆盖保存。

## 注意事项
- 仅放入真正跨项目复用且语义稳定的页面模板；项目私有页面放到项目存档 UI 源码目录。
- 动态变量必须使用占位符：`{{lv.<变量名>}}` / `{{ps.<变量名>}}`（仅普通自定义变量）。
