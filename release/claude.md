# `release/`

## 目录用途
- 本地打包/分发输出目录：用于放置从当前工作区生成的 release 压缩包（zip 等）。

## 当前状态
- 除 `claude.md` 外，其余内容均视为可重复生成的发布产物/临时工作区；默认不入库，可随时清理重建。
- 本目录在多数情况下可能是“空目录 + claude.md”，属于正常状态。

## 注意事项
- 不要在这里放任何源代码或需要长期保留的文件；只保留可清理的产物。
- 打包时建议排除 `__pycache__/`、`*.pyc`、`.pytest_cache/`、`app/runtime/cache/` 等运行期缓存/噪音文件。
