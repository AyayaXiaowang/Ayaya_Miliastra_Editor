## 目录用途
图结构与节点图语义相关的纯逻辑工具：通用图算法、端口/常量判定、节点实现库指纹等，为布局、解析、校验与缓存提供统一口径。

## 当前状态
- `graph_algorithms.py`：拓扑排序、事件流收集等通用算法。
- `graph_utils.py`：节点图语义工具（流程口判定、常量判定、归一化等）。
- `node_defs_fingerprint.py`：节点定义指纹计算，用于节点库/图缓存失效。

## 注意事项
- 仅依赖引擎层模型与标准库，禁止依赖 UI / `app/*` / `plugins/*`。
- 流程口/流程边判定统一复用 `graph_utils.is_flow_port_name()`，避免多处口径漂移。
- 算法保持纯函数与可复现，不引入 I/O 或日志副作用；指纹计算不读取文件内容，仅基于稳定元信息。
