## 目录用途
- 集中存放“项目存档/存档包”的解析状态文档（`解析状态.md`），用于跟踪 `.gil/pyugc → 项目存档` 的导出/反编译/校验进度。
- 该目录属于解析工具侧（`ugc_file_tools`），不再把状态文档放进 `Graph_Generater/assets/资源库/`，避免编辑器工程与解析流程耦合。

## 当前状态
- 该目录内容**可为空**：解析状态是可重复生成的“视图”。
- 生成后结构为：每个存档包一个子目录：`<package_id>/解析状态.md`（由 `ugc_file_tools/commands/extract_gil_to_package.py` 自动刷新）。

## 注意事项
- 状态文档是可重复生成的“视图”；权威数据以对应存档包目录（项目存档）内的 `原始解析/report.json` 与资源文件为准。
- 不在此文件中记录修改历史，仅保持用途/状态/注意事项的实时描述。
