# py_ugc_converter 目录说明

## 目录用途
- 存放基于 Python 实现的原神 UGC `.gil/.gia` 文件转换工具代码。
- 提供一套独立于 C++ 实现的 Python 版解析与转换逻辑，便于在脚本环境中批量处理 UGC 存档。
- 主要能力：读取二进制 `.gil/.gia`，结合 `dtype.json` 的结构描述，将内容导出为可读 JSON。

## 当前状态
- 代码结构按功能拆分为二进制读写、类型描述（dtype）、UGC 数据建模以及命令行入口等模块，尽量保持清晰可维护。
- 依赖标准库，不假设或强制安装第三方依赖。
- 对重复字段（dtype 未标注为数组但在数据中多次出现）会自动提升为数组节点；实现上避免出现“数组嵌套数组”的异常形态。

## 注意事项
- 类型描述文件默认使用 `ugc_file_tools/builtin_resources/dtype/dtype.json`；必要时通过命令行参数显式指定其它路径。
- 维护本目录代码时保持模块划分清晰，避免将无关逻辑混入转换核心。
- 本文件不记录修改历史，仅描述目录用途、当前状态与注意事项。
