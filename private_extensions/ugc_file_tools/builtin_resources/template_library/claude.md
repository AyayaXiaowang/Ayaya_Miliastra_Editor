# ugc_file_tools/builtin_resources/template_library 目录说明

## 目录用途
- 存放“**可安全递归扫描**”的 `.gil` 模板样本集合，用作 `--template-library-dir` 输入，避免扫描到不确定/不完整的样本导致解析失败或覆盖口径漂移。
- 目标：把用于写回/差异报告的模板样本库与 `save/` 下的杂项样本区分开，便于复用与长期维护。

## 当前状态
- 该目录下的子目录按场景拆分（例如按包/作用域/来源），每个子目录只放少量“已确认可解析”的 `.gil`。
- 命名规范：统一使用 **ASCII + lower_snake_case**（必要时带 `01_..` 的序号前缀便于排序与脚本引用）。

## 注意事项
- 子目录应尽量保持“少而精”：只放必要的 `.gil`，避免把超大或未经验证的存档混进来。
- 工具输出仍默认落在 `ugc_file_tools/out/`；需要长期复用的样本再人工/脚本复制进来。
