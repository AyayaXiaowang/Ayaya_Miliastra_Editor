# 千星沙箱全新 AI 网页 UI 工具
这套工具专为 AI 辅助创作而生，核心功能如下：

    ### 1. 真正意义上的“所见即所得”（实时预览）
    * **实时预览**：在浏览器里看到什么样子，导进游戏里就是什么样子。
    * **多分辨率适配**：内置 1920×1080、1600×900 等多种游戏标准分辨率，一键切换，确保你的 UI 在任何屏幕下都不跑版。
    * **延时摄影模式**：可以像看动画一样，一层一层查看 UI 的堆叠过程，方便检查遮挡和层级关系。

    ### 2. 节点图无缝联动（UI与逻辑一体化）
    * **交互直达**：支持节点图联动，你可以直接跟 AI 说：“按下这个按钮触发下一关，按下那个按钮打开商店”。
    * **一键生成**：AI 可以在生成网页 UI 的同时，一起为你生成对应的节点图逻辑。UI 按钮自带交互语义，导入后直接生效，无需手动重新连线。

    ### 3. 游戏数据动态绑定
    * **控件组 ID 动态绑定**：支持界面控件组 ID 的动态绑定，方便在逻辑中灵活调用。
    * **动态文本与进度条**：网页里可以直接写上游戏变量（例如玩家的生命值、关卡的倒计时）。工具会自动将它们绑定到游戏内的真实数据上，血条、经验条、积分榜轻松搞定。

    ### 4. 复杂多状态 UI 轻松搞定
    * **多状态支持**：一个按钮的“可用/禁用”状态，或者一个弹窗的“多页签切换”，只需在网页里定义好不同状态的样式。
    * **自动打包**：工具会自动帮你把这些状态打包成游戏内可一键切换的多状态控件，无需再手动复制粘贴多份 UI。

    ### 5. 极简导出，操作便捷友好
    * **一键导出**：支持直接导出为游戏存档文件（`.gil`）或布局资产（`.gia`），操作极其便捷友好。
    * **智能补偿**：工具会自动帮你处理游戏引擎特有的阴影、边框、以及缩放时的字号补偿，免去繁琐的手动微调。
    * **批量处理**：支持一次性导入/导出多个页面的 UI，大幅提升制作效率。

    ### 6. 贴心的 AI 纠错助手
    * **智能诊断**：如果 AI 生成的网页有不符合游戏规范的地方（比如字号不合规、颜色可能看不清），工具会精准指出问题。
    * **一键修复包**：提供“一键复制 AI 修复包”功能。你只需把错误信息直接发回给 AI，AI 就能看懂并自己把代码改对，形成完美的闭环！
